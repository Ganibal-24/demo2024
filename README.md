1. Выполните базовую настройку всех устройств:
   
                  а. Соберите топологию согласно рисунку. Все устройства работают на OC Linux - Debian.

                  e. Пул адресов для сети офиса HQ - не более 64. Для IPv6 пропустите этот пункт.
